### 初始化node项目
```shell
npm init -y
#指定国内仓，加快下载速度，--unsafe-perm不一定需要
npm install --unsafe-perm --registry=https://registry.npm.taobao.org

# pnpm 安装
npm install pnpm -g
# ts安装
npm install typescript -g
# pnpm 加速
pnpm set registry https://registery.npm.taobao.org
```
